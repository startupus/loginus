## Режимы работы

- **`hybrid`** (по умолчанию) - API v2 → кэш → статические файлы (fallback)
- **`dynamic`** - только API v2, без статических файлов
- **`static`** - только статические файлы (fallback режим)

## Переключение режимов

Через переменную окружения:
```bash
VITE_I18N_MODE=hybrid   # по умолчанию (рекомендуется)
VITE_I18N_MODE=dynamic  # только API v2
VITE_I18N_MODE=static   # только статика
```
