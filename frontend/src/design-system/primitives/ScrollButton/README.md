# ScrollButton

Кнопка для горизонтальной прокрутки списков/каруселей. Использует приподнятый фон `bg-surface-elevated`, чёткую границу и явный цвет иконки, чтобы не зависеть от наследования `currentColor` от контейнеров.

## Props

- `direction`: `'left' | 'right'` — направление прокрутки.
- `ariaLabel`: `string` — текст для скринридеров.
- `onClick`: `() => void` — обработчик клика.
- `className?`: `string` — дополнительные классы.

## Почему отдельный компонент

- Исключаем хрупкость цвета: `Icon` получает явный `color={'rgb(var(--color-text-primary))'}`.
- Обеспечиваем контрастность: используем `bg-surface-elevated` и `border-border` для видимого отличия на разных фонах.
- Единый стиль и поведение в разных секциях.
