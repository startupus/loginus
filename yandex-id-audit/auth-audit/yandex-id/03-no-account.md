# Яндекс ID - Экран "Нет аккаунтов с этим номером"

**URL:** `https://passport.yandex.ru/auth/reg/portal`  
**Заголовок:** Авторизация  
**Скриншот:** `03-no-account.png`

## Структура страницы

### Header
- **Кнопка "Назад"** (слева) → `/auth/list?mode=edit`
- **Логотип Яндекс ID** (по центру) → `https://ya.ru`

### Основной контент

#### Иллюстрация
- **3D иллюстрация:** Летающая тарелка (UFO) с фиолетово-синим свечением снизу

#### Заголовок
- **H1:** "Нет аккаунтов с этим номером"
- **Подзаголовок:** "Можно зарегистрировать новый аккаунт"

#### Отображение номера
```tsx
<div className="phone-display">
  <span className="phone-number">(912) 345-67-89</span>
</div>
```

#### Кнопки действий
```tsx
<div className="action-buttons">
  <Button variant="secondary">
    Изменить номер
  </Button>
  
  <Button variant="outline" bordered>
    Создать новый аккаунт
  </Button>
</div>
```

### Footer
- **Текст:** "Яндекс ID — ключ от всех сервисов"
- **Ссылка:** "Узнать больше" → `https://id.yandex.ru/about`

## UX паттерны

- **Clear Messaging** — понятное сообщение о том, что аккаунта нет
- **Visual Feedback** — иллюстрация для привлечения внимания
- **Two Paths** — возможность изменить номер или создать новый
- **Phone Confirmation** — отображение введённого номера для подтверждения
