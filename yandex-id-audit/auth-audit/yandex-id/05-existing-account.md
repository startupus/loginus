# Яндекс ID - Экран "На этот номер уже есть аккаунты"

**URL:** `https://passport.yandex.ru/auth/reg/portal`  
**Заголовок:** Авторизация  
**Скриншот:** `05-existing-account.png`

## Структура страницы

### Header
- **Кнопка "Назад"** (слева)
- **Логотип Яндекс ID** (по центру)

### Основной контент

#### Заголовок
- **H1:** "На этот номер уже есть аккаунты"
- **Подзаголовок:** "Войдите в аккаунт или создайте новый с другим номером"

#### Отображение номера
```tsx
<div className="phone-display">
  <span className="phone-number">8 (995) 150-34-44</span>
</div>
```

#### Кнопки действий
```tsx
<div className="action-buttons">
  <Button variant="primary">
    Войти
  </Button>
  
  <Link href="/support/id/new-phone-authorization.html">
    Написать в поддержку
  </Link>
  
  <Button variant="secondary">
    Изменить номер
  </Button>
</div>
```

## Ключевые особенности

1. **Конфликт номеров** — система обнаружила, что номер уже зарегистрирован
2. **Два пути** — войти в существующий или изменить номер
3. **Поддержка** — прямая ссылка на поддержку для решения проблем
4. **Чёткое сообщение** — понятно объяснена ситуация

## UX паттерны

- **Clear Messaging** — понятное объяснение проблемы
- **Multiple Solutions** — несколько вариантов решения
- **Support Access** — быстрый доступ к поддержке
- **Phone Confirmation** — отображение номера для проверки
