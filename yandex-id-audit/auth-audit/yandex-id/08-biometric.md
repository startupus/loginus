# Яндекс ID - Экран биометрической авторизации

**URL:** `https://passport.yandex.ru/auth/webauthn-auth`  
**Заголовок:** Авторизация  
**Скриншот:** `08-biometric.png`

## Структура страницы

### Header
- **Кнопка "Назад"** (слева) → `/auth/add`
- **Логотип Яндекс ID** (по центру)

### Основной контент

#### Заголовок
- **H1:** "Вход по лицу или отпечатку"
- **Инструкция:** "Следуйте инструкциям в появившемся окне"

## Ключевые особенности

1. **WebAuthn API** — использует стандарт Web Authentication API
2. **Системный промпт** — браузер/ОС показывает нативное окно для биометрии
3. **Минималистичный UI** — только инструкция, сам процесс происходит в системном окне
4. **Универсальность** — поддерживает и Face ID, и Touch ID, и другие биометрические методы

## Технические детали

```tsx
// Используется WebAuthn API
const credential = await navigator.credentials.get({
  publicKey: {
    challenge: new Uint8Array(32),
    rpId: 'passport.yandex.ru',
    userVerification: 'required',
    allowCredentials: [...]
  }
});
```

## UX паттерны

- **System Integration** — интеграция с системными возможностями устройства
- **Clear Instructions** — понятное объяснение что делать
- **Minimal UI** — не отвлекает от системного процесса
- **Fallback** — всегда есть кнопка "Назад" для возврата

## Безопасность

- **Local Authentication** — биометрия проверяется локально на устройстве
- **Challenge-Response** — используется криптографический challenge
- **User Verification** — требуется подтверждение пользователя
