# План модернизации Loginus на основе Яндекс ID

## Матрица соответствия страниц

| Яндекс ID | Loginus | Статус | Приоритет |
|-----------|---------|--------|-----------|
| `/` (Главная) | `/` (Dashboard) | Нужно модернизировать | Высокий |
| `/personal` | `/personal` | Нужно создать | Высокий |
| `/security` | `/profile/security` | Нужно модернизировать | Критично |
| `/family` | `/family` | Нужно создать | Средний |
| `/pay` | `/pay` | Нужно создать | Средний |
| `/helpdesk` | `/support` | Нужно создать | Низкий |

## Приоритеты модернизации

### Приоритет 1 (Критично)
1. **Безопасность** (`/security`)
   - Обновление пароля
   - Управление устройствами
   - История активности
   - Внешние аккаунты
   - Способы восстановления

### Приоритет 2 (Высокий)
2. **Главная страница** (`/`)
   - Виджеты дашборда
   - Попап профиля
   - Быстрые действия

3. **Персональные данные** (`/personal`)
   - Документы (9 типов)
   - Адреса
   - Автомобили
   - Питомцы
   - Контакты
   - Публичные данные
   - Управление данными

### Приоритет 3 (Средний)
4. **Семья** (`/family`)
   - Семейная группа
   - Приглашение членов
   - Детские аккаунты
   - Роли

5. **Платежи** (`/pay`)
   - Способы оплаты
   - История платежей
   - Подписки

### Приоритет 4 (Низкий)
6. **Поддержка** (`/helpdesk`)
   - Чат поддержки
   - FAQ

## Ключевые улучшения на основе Яндекс ID

### 1. Информационная архитектура
- **Модульная организация**: Разделение на логические секции
- **Якорные ссылки**: Навигация внутри страницы
- **Описательные тексты**: Пояснения для каждой секции

### 2. Визуальная иерархия
- **Карточный дизайн**: Четкие границы секций
- **Типографика**: Правильная иерархия заголовков
- **Иконография**: Уникальные иконки для функций

### 3. UX паттерны
- **Progressive Disclosure**: Основное видно, детали по клику
- **Quick Actions**: Быстрые действия в карточках
- **Contextual Navigation**: Ссылки "Все [тип]" для детальных страниц

### 4. Компоненты для создания

#### Primitives (дополнить)
- Separator - разделитель элементов

#### Composites (создать)
- DataSection - секция данных с заголовком и описанием
- SeparatedList - список с разделителями
- WidgetCard - карточка виджета для дашборда
- PaymentCard - карточка платежа
- FamilyMemberCard - карточка члена семьи

#### Layouts (улучшить)
- PageTemplate - добавить поддержку якорных ссылок
- Sidebar - улучшить активное состояние
