# ТЗ: Страница Яндекс Пэй (Платежи)

## URL
https://id.yandex.ru/pay

## Описание
Страница управления платежными методами, картами и историей платежей. Демонстрирует четкую организацию финансовой информации.

## Структура страницы

### Секция Способы оплаты
- Карты (счетчик количества)
- Ссылка на `/pay/cards`

### Промо-блоки
- Яндекс Плюс (30 дней бесплатно)
- Баланс Сплита

### Секция История платежей
- Список последних платежей с деталями:
  - Тип (Подписка Плюса)
  - Описание
  - Сумма (−399 ₽, −100 ₽)
- Ссылка "Все платежи" → `/pay/history`

## План модернизации для Loginus

### Компоненты
- `PaymentMethodsSection` - секция способов оплаты
- `PaymentHistoryList` - список платежей
- `PaymentCard` - карточка платежа

### Типизация
```typescript
interface Payment {
  id: string;
  type: 'subscription' | 'purchase' | 'refund';
  description: string;
  amount: number;
  currency: string;
  date: Date;
  status: 'completed' | 'pending' | 'failed';
}
```
