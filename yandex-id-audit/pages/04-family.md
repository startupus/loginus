# ТЗ: Страница Семья

## URL
https://id.yandex.ru/family

## Описание
Страница управления семейной группой, членами семьи, ролями и возможностями.

## Структура страницы

### Заголовок
- "Семейная группа" (h1)
- Карточка текущего пользователя (Админ)

### Действия
- Кнопка "Пригласить близкого"
- Ссылка "Создать детский аккаунт"

### Секция Возможности группы
- Семейная оплата
- Плюс для близких
- Тариф Яндекс 360
- Роли
- Удалить группу

### Секция Питомцы
- Список питомцев
- Ссылка "Все питомцы" → `/personal/pets`

## План модернизации для Loginus

### Компоненты
- `FamilyGroupHeader` - заголовок с информацией о группе
- `FamilyMembersList` - список членов семьи
- `FamilyFeaturesList` - возможности группы
- `InviteMemberModal` - модальное окно приглашения
