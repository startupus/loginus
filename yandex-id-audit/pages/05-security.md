## Описание
Центральная страница управления безопасностью аккаунта. Критически важная страница с множеством настроек.

## Актуальные секции (из визуального аудита)

1. **Промо-блок** - "Максимум защиты: Ваш аккаунт надёжно защищён Ключом" (с иллюстрацией бонсай)
2. **Способ входа**:
   - Текущий способ: Одноразовый пароль
   - Способы восстановления
3. **Контроль доступа**:
   - События (вся активность за 180 дней)
   - Ваши устройства (6)
   - Выйти везде
4. **Доступ к вашим данным**:
   - Добавить внешние аккаунты (3)
   - Управлять доступами
   - Пароли приложений

## Структура страницы

### Промо-блок биометрии
- Заголовок: "Подключите безопасный вход по лицу или отпечатку"
- Описание безопасности
- Кнопка "Подключить"

### Секция Способ входа
- Текущий способ (Обычный пароль)
- Кнопка "Обновить пароль" (с информацией "Менялся 14 месяцев назад")
- Ссылка "Способы восстановления" → `/security/recovery-methods`

### Секция Контроль доступа
- Ссылка "События" → `/security/activity` (Вся активность за 180 дней)
- Ссылка "Ваши устройства" → `/security/devices` (На которых вы вошли в Яндекс, счетчик: 1)
- Кнопка "Выйти везде"

### Секция Доступ к вашим данным
- Ссылка "Добавить внешние аккаунты" → `/security/external-accounts`
- Ссылка "Управлять доступами" → `/personal/data-access`
- Ссылка "Пароли приложений" → `/security/app-passwords`

## План модернизации для Loginus

### Компоненты
- `BiometricPromo` - промо-блок биометрии
- `LoginMethodsSection` - способы входа
- `AccessControlSection` - контроль доступа
- `DataAccessSection` - доступ к данным

### Критичные функции
- Обновление пароля
- Управление устройствами
- История активности
- Внешние аккаунты
