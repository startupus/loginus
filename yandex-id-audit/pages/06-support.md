## Описание
Страница поддержки пользователей с чатом-ботом и историей обращений.

## Актуальные элементы (из визуального аудита)

1. **Заголовок** - "Поддержка"
2. **Кнопка** - "Выбрать сервис"
3. **Левая панель** - История чатов:
   - Яндекс ID (в сети) - сегодня 14:41
   - Поддержка Кинопоиска - 7.06.25
   - Поддержка умных устройств с Алисой - 9.06.24
   - История обращений: Go, Маркет, Еда (закрыто)
4. **Правая панель** - Активный чат:
   - Сообщения бота поддержки
   - Кнопки выбора темы: Яндекс ID, Плюс, Почта, Маркет, Диск, Другой сервис
   - Поле ввода сообщения
   - Кнопка "Отправить файлы"

## Структура страницы
- Чат поддержки
- FAQ раздел
- Формы обращений

## План модернизации для Loginus

### Компоненты
- `SupportChat` - компонент чата
- `FAQSection` - раздел часто задаваемых вопросов
- `ContactForm` - форма обращения
